1 Emacs Class by Merlin Corey

Emacs class notes, scripts, and examples.

Example configuration tree in ./dotemacs/ installable with sh installer.sh (installer.sh).

1.1 History

Merlin has given this class a couple of times at Null Space Labs.

Class Location Date
Intro to Emacs Null Space Labs 2.0 2016 May 14
Introduction to Emacs Null Space Labs 3.0 2017 November 25

2 Presentation

Emacs is used for presentation with org-present-mode.

Org files are exported to HTML as well and thus readable on merlincorey.github.io.
